## Deploy localy:

##### clone & Install dependencies

```sh
git clone https://github.com/Nave-Cohen/Web-Project.git
cd Web-Project
npm install
```

#### \*\* First run - important \*\*

- edit credintals on .env.template (located in root folder)
  - do not change MYSQL_DATABASE=todo_app
- rename .env.template -> .env

##### Run

```sh
npm start
```

##### Tests

```sh
npm test
```
